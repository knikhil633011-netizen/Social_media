# 📖 About

This project is a full-stack social media platform built using modern web technologies. It provides users with an interactive experience similar to popular social networking platforms by allowing them to create posts, interact with other users, and manage their profiles.

The project focuses on responsive design, authentication, clean UI, and scalable backend architecture.

---

# ✨ Features

- 🔐 User Authentication (Login / Signup)
- 👤 User Profile
- 📝 Create Posts
- ❤️ Like Posts
- 💬 Comment on Posts
- 🖼 Upload Images
- 📰 News Feed
- 📱 Fully Responsive Design
- ⚡ Fast User Experience
- 🔒 Secure Authentication
- 🌙 Modern UI
